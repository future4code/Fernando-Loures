Projeto Descrição

O Projeto Sistema Loja simula o web site de uma loja de equipamentos eletrônicos. Esse é trabalho desenvolvido junto ao Curso Labenu e utiliza como ferramenta apenas HTML e CSS.

O projeto Mata-Mosquito é um exercício realizado no curso "Curso Web Completo 2020" na plataforma Udemy, neste projeto utiliza-se HTML,CSS, Bootstrap e JavaScript.

Eu sou um desenvolvedor iniciante, estudante de Desenvolvimento Web Fullstack na Labenu. Embora iniciante na área de desenvolvimento de software, tenho formação em Engenharia Metalúrgica e mestrado em Engenharia de Materiais.
